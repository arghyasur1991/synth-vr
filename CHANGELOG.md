# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-04

### Added
- PlayerHandBodies: MuJoCo-bound hand physics from OVR hand tracking with grab mechanics
- PlayerLocomotion: smooth walk and rotate via controller thumbstick
- SceneMeshManager: MRUK room integration with physics colliders, furniture anchors, and occlusion
- Automatic Synth floor positioning with room-aware spawn point selection
- AmbientLightEstimator: passthrough camera-based lighting estimation
- PTRLWithDepth shader and occluder materials for mixed reality rendering
- Optimized ctrl callback with cached transforms for VR frame budgets
